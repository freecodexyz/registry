# $freecode token

::: info Live
The `$freecode` token is live, see on [Dexscreener](https://dexscreener.com/base/0x67a7ca081dc79b45fd1fa059cd3b8dcca779aba3).
:::

## Intended role

In the FCF stack as designed, `$freecode` is the unit of account for protocol-level cashflow. The plan is for it to be the medium through which the layer above RIK pays into and out of the repositories it indexes:

- **Funding** routed to a RIK holder.
- **Rewards** distributed against RIK holders for specific protocol-defined behavior.
- **Staking** of `$freecode` against a repo, with the RIK as the identity being staked against.
- **Trading fees** on the future code marketplace.
